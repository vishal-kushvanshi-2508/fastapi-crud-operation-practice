# FastAPI CRUD Operations Practice

## 📖 Overview

This project is a Python-based REST API developed using FastAPI to perform CRUD (Create, Read, Update, Delete) operations. The project demonstrates modern backend development practices, API design, request handling, data validation, and database interaction.

It serves as a hands-on implementation of FastAPI fundamentals and RESTful API development.

---

## 🚀 Features

* Create new records
* Retrieve single and multiple records
* Update existing records
* Delete records
* Request validation using Pydantic
* RESTful API architecture
* Error handling and response management

---

## 🛠️ Technologies Used

* Python
* FastAPI
* Pydantic
* Uvicorn
* MySQL / SQLite
* REST API

---

## 📊 API Operations

### Create

* Add new records

### Read

* Get all records
* Get record by ID

### Update

* Modify existing records

### Delete

* Remove records
